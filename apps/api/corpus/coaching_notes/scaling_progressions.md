# Scaling Progressions and Coaching Notes

## Scaling Philosophy

Scale to preserve the intended stimulus of the workout, not just to "finish." A WOD scaled correctly should take approximately the same time window as RX (e.g., Fran: 2-8 minutes). If the scaling keeps someone working for 20+ minutes when the stimulus is "sprint," scale further.

## Pull-up Progressions (Weakest → Strongest)

1. Ring rows (body angle = difficulty)
2. Banded pull-ups (heavy band → light band)
3. Jumping pull-ups (kipping practice)
4. Kipping pull-ups
5. Butterfly pull-ups (for cycling speed)
6. Strict pull-ups (more demanding, better for strength)

## Thruster Scaling

- Reduce load first (always preserve full range of motion)
- PVC → empty bar → 65/45 lb → 75/55 lb → 95/65 lb (RX)
- If shoulder mobility limits overhead position, DB thrusters allow more natural wrist rotation
- If squat depth is the limiter, goblet squat + press separately as warmup drill

## Handstand Push-up Progressions

1. Pike push-up (on floor)
2. Box pike push-up (elevate feet)
3. Wall-facing HSPU with abmat
4. Kipping HSPU
5. Strict HSPU
6. Deficit HSPU (advanced)

## Double-Under Progressions

1. Single-unders (3:1 ratio for workout substitution)
2. Penguin jumps (practice rope timing without rope)
3. Attempts (1 DU + single + 1 DU)
4. Consecutive DUs
   If an athlete can do 30+ DUs unbroken in isolation but breaks up during WODs, the limiter is aerobic tolerance, not skill.

## Squat Clean Scaling

- Romanian DL → hang power clean → power clean → squat clean
- Use submaximal weight and reinforce receiving position before adding load
- "Elbows to the ceiling" cue for front rack position

## Common Coaching Cues by Movement

### Deadlift

- "Chest up, hips back" for setup
- "Protect your armpits" (cue for lat engagement)
- "Push the floor away" instead of "pull the bar up"

### Snatch

- "Bar stays close — like you're zipping up a jacket"
- "Jump and shrug" for extension
- "Punch up and back" for receiving position

### Kipping Pull-up

- "Arch and hollow" — teach the swing first
- "Late kip" — don't pull too early; wait for the swing
- "Elbows back" at the top to clear the chin

### Box Jump

- "Load and explode" — quick dip, immediate jump
- "Stick the landing" — practice controlled landing before adding height
- Step down always for high volume (>30 reps) to protect Achilles
